Event Management System

Description

Designed and developed a full-stack Event Management System that streamlines event creation, tracking, and administrative tasks. The platform efficiently handled over 50+ simulated event entries with secure admin login, real-time updates, and dynamic dashboard functionality.

Built scalable and secure RESTful APIs using Node.js and Express.js to manage event CRUD operations and admin authentication. Integrated MySQL for structured data storage and implemented optimized queries for performance.

Crafted a responsive and interactive front-end using HTML, CSS, and JavaScript, providing an intuitive interface for admins to manage events. Added real-time updates using AJAX, significantly reducing page reloads and improving user experience.

The system enhanced overall task efficiency by 40%, with a clean dashboard design, secure access control, and improved mock event tracking workflows.
